В 3х папках расположены соответствующе файлы.
Необходимые файлы для дз имеют Hw в конце названия;

Компаратор для учителей основывается на показателях з-п;
Сервисы для всех пользователей позволяют создавать (т.к параметры не идентичны, в конструкторах описана логика создания
    только по совпадающим полям), просматривать весь список учителей в системе и редактировать конкретный экземпляр
    того или иного класса.
Реализация Вью-компонента для учителей аналогично студентам.
