Игра "Быки и коровы"

Класс Main - точка входа;

Папка typeOfGames содержит в себе:
 -интерфейс игра;
 -абстрактный класс, реализующий методы интерфейса;
 -конструкторы 3 типов игр;
 -enum статусов игры;

Класс Answer подразумевает собой ответ пользователя, который храниться в логах (logs/application.txt);

Класс GameInitialization содержит в себе методы:
 -начало игры;
 -реализации выбора типов игры и ее прохождения;
 -перезапуск игры;
